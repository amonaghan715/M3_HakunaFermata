Chat GPT 5.1 was used in the creation of this project as a resource on what scales and sounds to use to achieve specific effects. This included the idea to use the
lydian scale to achieve “magical” and  “bright” tones, and how to achieve a brown noise effect.

Prompts used:
- What musical scale would achieve a bright, magical tone?
- What is brown noise compared to white noise?
